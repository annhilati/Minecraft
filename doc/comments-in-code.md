# Comments

### GLSL (.glsl; .fsh; .vsh)
```glsl
/*
Comment
*/

code{
	variable = value();
}

// Another Comment //
```

### Java properties (.properties)
```properties
variable=value
# Comment
```

### JSON (.json)
JSON doesnt support comments. Its data only

### YAML (.yml)
```yaml
variable: value
# Comment
```
