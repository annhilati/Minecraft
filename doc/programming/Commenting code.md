# Comments

### GLSL (.glsl; .fsh; .vsh; .settings)
```glsl
/*
Comment
*/

code{
	option = value();
}

// Another Comment //
```

### Java properties (.properties)
```properties
option=value
# Comment
```

### JSON (.json; .mcmeta)
JSON doesnt support comments. Its data only

### Language (.lang)
```
option=value
/ Comment
```

### Minecraft function (.mcfunction)
```mcfunction
command to do something
# Comment
```

### YAML (.yml)
```yaml
option: value
# Comment
```
